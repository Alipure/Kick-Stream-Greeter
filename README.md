# Kick-Stream-Greeter
For educational purposes.

The motive for this program is to greet every live streamer automatically

Installation (WINDOWS):
```
Install Node -  https://nodejs.org/en/download
```


In the folder's terminal:

Install NPM
```
powershell.exe -ExecutionPolicy Bypass -File "C:\Program Files\nodejs\npm.ps1" install -g npm
```

Install Selenium Webdriver
```
powershell.exe -ExecutionPolicy Bypass -File "C:\Program Files\nodejs\npm.ps1" install selenium-webdriver
```


Install chromedriver
```
powershell.exe -ExecutionPolicy Bypass -File "C:\Program Files\nodejs\npm.ps1" install chromedriver
```
